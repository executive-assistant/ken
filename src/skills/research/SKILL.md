# Research Skill

You have expertise in gathering, analyzing, and synthesizing information from various sources.

## Capabilities

### Web Search
- Formulate effective search queries
- Evaluate source credibility
- Cross-reference information
- Track and cite sources

### Documentation Analysis
- Extract key information from technical docs
- Identify relevant sections quickly
- Summarize complex topics
- Create reference notes

### Information Synthesis
- Combine information from multiple sources
- Identify patterns and connections
- Present balanced perspectives
- Distinguish facts from opinions

## Research Process

1. **Define the question** - Clarify what you're looking for
2. **Search broadly** - Use multiple queries and sources
3. **Evaluate sources** - Check credibility and recency
4. **Extract key points** - Note important information
5. **Synthesize findings** - Combine into coherent summary
6. **Store in memory** - Save for future reference

## Output Format

When presenting research:

```markdown
## Summary
[Brief overview of findings]

## Key Findings
- Point 1
- Point 2
- Point 3

## Sources
- [Source 1](url)
- [Source 2](url)

## Recommendations
[If applicable]
```

## Tips

- Always check the date of information
- Prefer official documentation over third-party sources
- Note any conflicting information
- Save research to /memories/ for future reference
